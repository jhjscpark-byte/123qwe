## 로컬에서 보기
1. 리포지토리를 클론하거나 파일을 다운로드합니다.
   git clone https://github.com/jhjscpark-byte/123qwe.git
2. 다운로드한 `migraineslide` 파일을 같은 폴더에 두고 웹 브라우저로 엽니다(브라우저에서 파일을 드래그하거나 `file://` 경로로 열기).

## GitHub Pages로 호스팅(선택)
원한다면 리포지토리에서 GitHub Pages를 활성화하면 더 안정적으로 호스팅할 수 있습니다.
- Settings → Pages → Source에서 브랜치(예: main)와 루트(/)를 선택하고 저장
- 활성화 후 제공되는 URL로 접속하면 HTML 파일을 호스팅할 수 있습니다(파일명과 경로에 따라 URL 조정 필요).
